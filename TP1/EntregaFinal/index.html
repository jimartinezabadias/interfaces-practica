Filtros

Los filtros no son acumulables, pero si se dibuja o borra sobre una imagen filtrada se puede volver a aplicar filtros.

Tamaño de imágenes

Las imágenes que superan el tamaño máximo ( 800 px x 552 px ) se reducen manteniendo sus proporciones. La imagen editada que se descarga esta reducida si supera los máximos.
